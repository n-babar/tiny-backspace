# Simple API Example

A simple Flask API with POST endpoints for users, posts, and comments.

## Endpoints

- `POST /users` - Create a new user
- `POST /posts` - Create a new post  
- `POST /comments` - Create a new comment

## Running

```bash
pip install -r requirements.txt
python app.py
```
